* Python的特点

~~~Markdown
主要特点：
1. 语法简洁清晰，优雅
2. 有非常强大的类库（模块）
		1. 模块多
		2. 模块功能强
3. 胶水语言
		可以使用别的语言的类库：C  Java
4. 纯面向对象的语言
		方法是对象，函数是对象，文件是对象，包是对象，模块是对象
		万物皆对象

其他特点：
1. 跨平台性
		平台：操作系统  windows  Linux  OSX
2. 把源代码编译成字节码文件
		利用编译器
		字节码：bytecode---本质是一个二进制文件（0,1）---计算机不能直接执行
		字节码文件是让虚拟机执行（字节码文件是虚拟机指令）
3. 代码量少
		Python代码量大约是Java的五分之一
~~~

* Python的运行机制

~~~markdown
运行：程序员---源代码   计算机---0101  
向计算机发号施令
1. 编译型：
		将源代码通过编译器，编译成机器码文件（0101）---可以直接在计算机上运行的代码
		优势：执行效率高
		弊端：不能跨平台
2. 解释型：
		将源码通过解释器，直接解释运行（不产生任何其他文件）
		逐行解释翻译
		弊端：执行效率低
		优势：可以跨平台

3. Python的执行机制：
		1. 以脚本的形式执行（纯解释型）---脚本语言
		2. 先编译，后解释执行
			编译：字节码文件（二进制文件）---只能在虚拟机中执行
			解释：虚拟机就是解释器（虚拟机中包含了解释器）
			源文件-编译器-字节码文件-虚拟机（软件）-操作系统
			
			优势：
				1. 相对于解释型效率高
				2. 可以跨平台
~~~

* Python的实现

* 编写第一个程序：hello world

~~~markdown
1. 交互式解释器：
		交互式界面是一种解释器

		2. IDLE：交互式解释器
~~~
