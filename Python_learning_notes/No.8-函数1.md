* 函数的参数

~~~markdown
参数的划分：
1. 形式参数：Parameter---简称形参
		创建函数时，小括号内的参数
		相当于函数内部的一个局部变量
		传值的过程相当于赋值的过程
		1. 没有定义任何参数：无参
2. 实际参数：Argument---简称实参
		调用函数时，小括号内传递的参数
		一定是一个确定的值
注意：
1. 如果定义多个形参，要用逗号隔开
2. 所有的形式参数，在一次调用中只能被赋值一次
3. 函数的定义中，形参只能出现一次
~~~

* 位置参数

~~~markdown
根据参数的位置，一一对应的传递参数（相对于形参和实参）
1. 如果定义了形式参数，则必须给出实际参数
2. 参数必须一一对应，不能多也不能少
3. 形参和实参之间注意顺序
~~~

~~~python
def fun(a,b,c):
    # a和b应该是个字符串， c是个列表
    print(a,b,c)

fun(1,2,3)
~~~

* 关键字参数

~~~markdown
定义了关键字的参数（相对实参）
1. 根据实参中定义的关键字，和形参中对应的参数一一对应
2. 不能多也不能少
3. 关键字参数不能重复的给某个形参赋值
~~~

~~~python
def fun(a,b,c):
    print(a,b,c)
fun(c=3,b=2,a=1)

def fun(a,b):
    pass
fun(a=1,a=2)  # 错误
~~~

* 默认参数

~~~markdown
定义了默认值的参数（相对形参）
1. 默认参数会在形参的位置设置默认值
2. 如果传递了实际参数，则使用对应的实际参数
3. 如果没有传递实际参数，则使用默认参数
4. 注意：默认参数后不能出现位置参数
~~~

~~~python
def fun(a=10,b=20,c=30):
    print(a,b,c)
fun(1,2,c=3)
fun()
~~~

* 可变长参数

~~~markdown
也称为收集参数
可以收集多个参数
区分于序列解包

1. *args
	将参数获得，并打包成一个元组
	实参以位置参数传递（只支持位置参数传值）
2. **kwargs
	只支持使用关键字参数传值，并将结果打包成字典，切关键字作为字典的键，关键字对应的值作为字典的值
	
注意：可变长参数在定义中每种只能出现一个
~~~

~~~python
def fun(*a):
    print(a) # 元组
    for i in a:
        print(i)

fun(1,2,3,4,5)

==============================
def fun(**kwargs):
    print(kwargs)

fun(a=1,b=2,c=3)

~~~

* 混用形式

~~~markdown
1. 关键字参数+位置参数：
		位置参数只能出现在关键字参数之前
		原因：编译器认为参数中第一次出现=号的形式，之后的所有参数都应该有=号
		1. 不能重复给某一个形参赋值
		
2. 位置参数+可变长参数：
		1. 位置参数可以写在可变长参数之后，但是此时必须通过关键字参数传递实参
3. 默认参数+可变长参数：
		没有具体要求
~~~
